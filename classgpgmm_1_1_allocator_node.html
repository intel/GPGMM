Detailed Description

template<typename T>
class gpgmm::AllocatorNode< T >

AllocatorNode stores one or more allocators in a intrusive LinkedList.

AllocatorNode can also be created with another AllocatorNode. AllocatorNode represents a chain where allocations made between the first-order AllocatorNode (or parent) and the next AllocatorNode (or child) form a one-way edge.
