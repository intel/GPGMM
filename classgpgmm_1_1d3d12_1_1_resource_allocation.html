Maps the resource allocation.

Gets the CPU pointer to the specificed subresource of the resource allocation.

If sub-allocated within the resource, the read or write range and pointer value will start from the allocation instead of the resource.

Parameters
subresourceSpecifies the index number of the subresource.
pReadRangeA pointer to a D3D12_RANGE structure that describes the range of memory to access.
[out]ppDataOutA pointer to a memory block that receives a pointer to the resource data.

Implements gpgmm::d3d12::IResourceAllocation.
