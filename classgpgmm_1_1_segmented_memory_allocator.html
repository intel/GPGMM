◆ TryAllocateMemory()

std::unique_ptr< MemoryAllocation > gpgmm::SegmentedMemoryAllocator::TryAllocateMemory ( const MemoryAllocationRequest &  request)
overridevirtual

Create a memory allocation.

Creates a MemoryAllocation that has at-least requested size whose value is a multiple of the requested alignment. If it cannot, return nullptr. The returned allocation is only valid for the lifetime of MemoryAllocator.

Parameters
requestA MemoryAllocationRequest to describes what to allocate.
Returns
A pointer to MemoryAllocation. If NULL, the request could not be full-filled.
