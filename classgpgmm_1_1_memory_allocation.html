Detailed Description

Represents a location and range in memory.

It can represent a location in memory by one of two ways: 1) a range within a memory block or 2) a memory block of the entire memory range.

A sub-allocated memory allocation (or sub-allocation) is a non-zero offset, non-overlapping range, in memory.

Parameters

memoryA pointer to the underlying MemoryBase that will contain the memory allocation.

methodThe method to describe how the allocation was created.
